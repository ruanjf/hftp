hftp
====

a simple static http or ftp server

`hftp <dir> <port> <tpye>`

`hftp d:\tt 8888 http`

 - 用法

  - `dir` 指定服务的文件夹（绝对路径）
	- `port` 指定服务的端口，默认值`8888`
 	- `type` 指定服务的类型（http、ftp），默认值`ftp`

 - Use

	- `dir` service folder(Absolute path)
	- `port` service port, default`8888`
 	- `type` service type (http、ftp), default`ftp`
